## 🤝 AIは相棒である
---

生成AIを使いこなしている人の多くは、AIに「正解」を求めていません。
自分の頭の中にあるモヤモヤした考えを**外に出して、構造化して、検討する**ためにAIを使っています。

つまり、AIは「賢い辞書」ではなく「考えを壁打ちできる相棒」です。

この考え方を前提にすると、AIの活用の幅が大きく広がります。
以下に、4つの実践的な使い方を紹介します。

<br>

## 💡 4つの実践的な使い方
---

### - やりたいことを企画書にしてもらう

漠然としたアイデアを構造化するのはAIが得意な仕事です。
「何かやりたい」というモヤモヤを、まずAIに投げてみましょう。

```
学内で生成AIの活用を推進したいと思っています。
まだ漠然としたアイデアですが、企画書の形に整理してもらえますか？
対象は教職員で、予算は限られています。
```

{{% alert icon="🤖" context="success" %}}
以下に企画書のたたき台を作成しました。

**企画名**: 教職員向け生成AI活用推進プログラム

**目的**: 教職員が業務において生成AIを効果的に活用できるようにし、業務効率化と教育・研究の質向上を図る

**対象**: 全教職員（段階的に展開）

**実施内容**:
1. 基礎研修（2時間 x 2回）：生成AIの基本操作とリスク理解
2. 実践ワークショップ（月1回）：業務別の活用事例を共有
3. 相談窓口の設置：日常的な疑問に対応

**スケジュール**: 第1四半期に基礎研修、第2四半期から実践ワークショップを開始

**予算概算**: 外部講師費用を抑え、内部人材を活用する方針で約30万円
{{% /alert %}}

大事なのは、この出力をそのまま使うことではありません。
**「自分は何をやりたかったのか」が企画書の形で見える化される**ことで、足りない視点や次のアクションが明確になります。

<br>

### - いろいろな立場の声を出してもらう

企画や提案を考えるとき、自分一人の視点だけでは見落としが生じます。
AIに複数の立場からの意見を出してもらうことで、多角的な視点を得ることができます。

```
以下の企画について、3つの異なる立場から意見を出してください。
- 学生の立場
- 教員の立場
- 事務職員の立場

企画：授業の課題提出にAIの利用を認めるガイドラインを作成する
```

{{% alert icon="🤖" context="success" %}}
**学生の立場**:
AIを使えること自体は歓迎しますが、「どこまで使っていいのか」の基準が曖昧だと不安です。具体的なOK/NGの例を示してほしいです。またAIを使える学生と使えない学生の間で不公平が生じないか心配です。

**教員の立場**:
AIを使った課題と使わない課題の評価基準をどう分けるかが課題です。学生の学びの質をどう担保するか、AIに頼りすぎて基礎力が身につかないリスクも考慮すべきです。一方で、AIを適切に使う能力も今後の社会では重要です。

**事務職員の立場**:
ガイドラインの運用・管理の負荷が気になります。問い合わせ対応のフローを明確にしてほしいです。また学部ごとに運用が異なると混乱するため、全学的な統一基準が望ましいと考えます。
{{% /alert %}}

自分が思いもしなかった観点が見つかることがあります。
**AIに多様な立場を演じてもらう**ことで、一人で考えるよりも検討の幅が広がります。

<br>

### - 情報を行動に圧縮してもらう

長い報告書や議事録を読んだあと、「で、結局何をすればいいの？」と思うことはありませんか。
AIに情報を渡して「次に取るべきアクション」に変換してもらいましょう。

```
以下の会議メモを読んで、私が来週までにやるべきことを
優先度順にリストアップしてください。

（会議メモをここに貼り付け）
```

{{% alert icon="🤖" context="success" %}}
来週までのアクションリスト（優先度順）:

1. **【高】予算案の修正版を作成** - 田中部長の指摘を反映し、金曜日までに再提出
2. **【高】外部講師候補への打診** - 3名の候補にメールで日程調整の連絡
3. **【中】アンケート項目の素案作成** - 次回会議で議論するためのたたき台
4. **【低】過去の類似事例の収集** - 他大学の取り組みを3件程度調べておく
{{% /alert %}}

膨大な情報を「行動」に圧縮する作業は、人間にとって意外と負荷が高いものです。
AIに一次整理を任せることで、**自分は判断と実行に集中できます。**

<br>

### - 無難な案をあえて出してもらう

AIは「平均的な回答」を出すのが得意です。これを逆手に取りましょう。
まずAIに無難な案を出してもらい、それを踏み台にして**自分ならではの案を考える**のです。

```
大学のオープンキャンパスで来場者に配るノベルティグッズのアイデアを
10個出してください。一般的なものでかまいません。
```

{{% alert icon="🤖" context="success" %}}
1. クリアファイル（大学ロゴ入り）
2. ボールペン
3. トートバッグ
4. ステッカー
5. メモ帳
6. うちわ（夏季向け）
7. マスキングテープ
8. ポストカード（キャンパス写真）
9. 缶バッジ
10. ミニタオル
{{% /alert %}}

AIの回答は予想通り「無難」なリストです。しかしここからが本番です。
「どれも見たことがある。自分たちの大学ならではの特徴を活かすとしたら？」と考え始めることで、**既存の枠組みから一歩踏み出した発想が生まれやすくなります。**

AIの無難さは弱点ではなく、**創造的な思考のための出発点**として使えます。

## 🔑 一番大切なこと
---

4つの使い方に共通する考え方があります。

{{% alert icon="" context="warning" %}}
<strong>AIに任せている部分と、自分で判断している部分の境界に常に意識を向けること</strong>
{{% /alert %}}

AIは情報の整理・構造化・多角的な視点の提示が得意です。
一方で、最終的に**何を選び、何を捨て、どう行動するかは人間の仕事**です。

この境界が曖昧になると、AIに依存した意思決定に陥ります。
逆にこの境界を意識し続けることで、AIとの協働は**あとからじわじわと効いてくる力**になります。

「AIを使いこなす」とは、AIに何でもやらせることではありません。
**AIとの対話を通じて、自分自身の考えをより良くしていくこと**です。
