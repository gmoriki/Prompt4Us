## プレスリリースを作成する
---

{{< alert context="info" text="**対象: 広報担当者**<br>**想定シーン**: 広報課の職員が、本学教員の研究成果が学術誌に掲載されたことを受け、報道機関向けのプレスリリースを作成する。" />}}

{{< alert context="warning" text="引用元: [Columbia University CTL Prompt Library](https://ai.ctl.columbia.edu/explore/prompts/) を参考に作成" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学広報のプレスリリース作成に精通したライターです。
以下の情報をもとに、報道機関向けのプレスリリースを作成してください。

# 発表内容
- 大学名：【変更箇所：〇〇大学】
- 発表タイトル：【変更箇所：〇〇学部 △△教授の研究グループが、□□に関する新たなメカニズムを解明】
- 研究概要：【変更箇所：△△教授らの研究グループは、□□の分子メカニズムを世界で初めて解明した。この発見は、将来的に◇◇の治療法開発に貢献する可能性がある。】
- 掲載誌：【変更箇所：Nature Communications（2026年2月号）】
- 発表日：【変更箇所：2026年2月13日】

# プレスリリースの構成要件
1. タイトル（インパクトがあり、内容が端的に伝わるもの）
2. サブタイトル（研究のポイントを1文で）
3. リード文（5W1Hを含む200字程度の要約）
4. 研究の背景（なぜこの研究が必要だったか）
5. 研究成果の概要（専門用語を避け、一般読者にもわかる表現で）
6. 今後の展望
7. 研究者コメント
8. 論文情報（タイトル、著者、掲載誌、DOI）
9. 問い合わせ先

# 注意事項
- 専門用語には簡潔な補足説明を添えること
- 正確性を重視しつつも、一般メディアの記者が記事化しやすい文体にすること
- 全体で1,200〜1,500字程度
```

**出力例**

**プレスリリース**

**〇〇大学 △△教授らが□□の新たなメカニズムを世界初解明**
― 将来の◇◇治療法開発への貢献に期待 ―

〇〇大学（所在地：〇〇県、学長：〇〇）〇〇学部の△△教授らの研究グループは、□□における分子レベルのメカニズムを世界で初めて解明しました。この研究成果は、国際学術誌「Nature Communications」（2026年2月号）に掲載されました。

**【研究の背景】**
□□は従来...（以下、背景の説明が続く）

**【研究成果の概要】**
本研究では...（以下、成果の説明が続く）

**【今後の展望】**
本成果を基盤として...（以下、展望が続く）

**【問い合わせ先】**
〇〇大学 広報課
TEL：XXX-XXX-XXXX
E-mail：press@example-univ.ac.jp

```markdown
あなたは大学広報のSNS運用に精通したマーケターです。
以下の情報をもとに、SNS投稿文を作成してください。

# 投稿内容
- 大学名：【変更箇所：〇〇大学】
- 告知内容：【変更箇所：夏のオープンキャンパス開催（2026年8月3日・4日）】

1. Twitter/X用（140字以内）を3パターン
2. Instagram用キャプション（300字程度）を1パターン

- 日時・場所の情報は正確に記載
```

## SNS投稿の炎上リスクをチェックする
---

{{< alert context="info" text="**対象: 広報担当者**<br>**想定シーン**: 広報担当の職員が、SNSに投稿する文章を公開前にチェックし、炎上や批判を招く表現がないか確認したい。" />}}

{{< alert context="warning" text="引用元: [東京都デジタルサービス局「文章生成AI活用事例集」](https://www.digitalservice.metro.tokyo.lg.jp/documents/d/digitalservice/ai_prompt/) を参考に作成" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学広報のリスク管理に精通した専門家です。
以下のSNS投稿文について、炎上リスクの観点からチェックしてください。

# チェック対象の投稿文
【変更箇所：ここにチェックしたいSNS投稿文を貼り付けてください】

# 投稿先
【変更箇所：Twitter/X】

# チェック観点
1. 差別・偏見につながる表現がないか
2. 特定の個人・団体を傷つける表現がないか
3. 事実誤認や誤解を招く表現がないか
4. 著作権・肖像権に抵触する可能性がないか
5. 機密情報や個人情報の漏洩リスクがないか
6. ジェンダー・多様性への配慮が不足していないか
7. 社会的に敏感なトピックへの不適切な言及がないか
8. 大学の公式アカウントとしてふさわしいトーンか

# 出力形式
1. リスク評価（高・中・低・なし）の総合判定
2. 各チェック観点ごとの評価結果（問題あり/注意/問題なし）と具体的な指摘
3. 問題がある場合は修正案を提示
4. 投稿前に追加で確認すべき事項があれば記載
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**SNS投稿文 炎上リスクチェック結果**

**総合リスク評価：中**

| チェック観点 | 評価 | 指摘事項 |
|------------|------|---------|
| 差別・偏見 | 注意 | 「女子学生に人気の学部」という表現がジェンダーステレオタイプを助長する可能性あり |
| 個人・団体への影響 | 問題なし | ― |
| 事実誤認 | 問題なし | ― |
| 著作権・肖像権 | 注意 | 添付画像に学生の顔が映っている場合、掲載許可の確認が必要 |
| 機密情報・個人情報 | 問題なし | ― |
| ジェンダー・多様性 | 注意 | 上記の差別・偏見と同様 |
| 敏感なトピック | 問題なし | ― |
| トーンの適切性 | 問題なし | ― |

**修正提案**
- 修正前：「女子学生に人気の〇〇学部」
- 修正後：「幅広い学生から支持される〇〇学部」

**追加確認事項**
- 添付画像に写っている人物の掲載許可を取得しているか確認してください

{{% /alert %}}


## Webページのコンテンツを作成する
---

{{< alert context="info" text="**対象: 広報担当者**<br>**想定シーン**: 広報課・学部事務の職員が、大学Webサイトに掲載する学科紹介ページのコンテンツを新規作成・リニューアルすることになった。" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
# 学科の基本情報
- 大学名：【変更箇所：〇〇大学】

- 学科の特色：【変更箇所：少人数ゼミによる実践的な教育、地域フィールドワークの充実、企業との連携プロジェクト】
- 取得可能な資格：【変更箇所：〇〇士、△△検定】
- 卒業後の進路：【変更箇所：〇〇業界、△△分野、大学院進学】

1. ページの見出し構成（H2・H3レベル）
```

## 多言語キャッチコピーを作成する
---

{{< alert context="info" text="**対象: 広報担当者**<br>**想定シーン**: 広報課の職員が、大学のブランディングに使用するキャッチコピーを日本語・英語・中国語の3言語で作成したい。パンフレットやWebサイトで統一的に使用する予定。" />}}

{{< alert context="warning" text="引用元: [東京都デジタルサービス局「文章生成AI活用事例集」](https://www.digitalservice.metro.tokyo.lg.jp/documents/d/digitalservice/ai_prompt/) を参考に大学文脈に改変" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは多言語コピーライティングに精通したクリエイターです。
以下の情報をもとに、大学ブランディング用のキャッチコピーを作成してください。

# 大学の基本情報
- 大学名：【変更箇所：〇〇大学】
- 建学の精神：【変更箇所：「自主自律」】
- 大学の強み：【変更箇所：実学教育、国際交流、地域貢献】
- ブランドイメージ：【変更箇所：伝統と革新の融合、学生一人ひとりに寄り添う教育】

# 使用場面
【変更箇所：大学パンフレット、Webサイト、SNS、オープンキャンパスの看板】

# 出力要件
1. キャッチコピーを5案作成
2. 各案について以下を記載：
   - 日本語版
   - 英語版
   - 中国語（簡体字）版
   - コピーの意図・ねらいの説明（日本語で100字程度）
3. 各言語で自然な表現とし、直訳ではなくローカライズすること

# 注意事項
- 短く覚えやすいフレーズにすること（各言語15語以内を目安）
- 文化的に不適切な表現がないよう配慮すること
```

## オープンキャンパスの告知文を作成する
---

{{< alert context="info" text="**対象: 広報担当者**<br>**想定シーン**: 入試課・広報課の職員が、オープンキャンパスの告知文をWebサイトに掲載するために作成する。高校生と保護者の両方に情報がわかりやすく伝わる構成にしたい。" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学のイベント広報に精通したライターです。
以下の情報をもとに、Webサイト掲載用のオープンキャンパス告知文を作成してください。

# イベント情報
- 大学名：【変更箇所：〇〇大学】
- イベント名：【変更箇所：夏のオープンキャンパス2026】
- 開催日時：【変更箇所：2026年8月3日（日）・4日（月） 10:00〜16:00】
- 会場：【変更箇所：〇〇キャンパス（〇〇県〇〇市〇〇町1-1）】
- アクセス：【変更箇所：〇〇駅から徒歩10分、無料送迎バスあり】
- 対象：【変更箇所：高校生（全学年）、保護者、高校教員】
- 事前予約：【変更箇所：要予約（大学Webサイトから申込み）】

# 主なプログラム
- 【変更箇所：学部別模擬授業（全8学部）】
- 【変更箇所：在学生によるキャンパスツアー】
- 【変更箇所：入試相談ブース】
- 【変更箇所：奨学金・学生生活相談コーナー】
- 【変更箇所：学食無料体験】

# 出力要件
1. ページタイトルとリード文（150字程度）
2. プログラム紹介（各プログラムの簡潔な説明付き）
3. 参加方法・申込案内
4. アクセス情報
5. よくある質問（FAQ）3〜5件
6. 問い合わせ先

# 注意事項
- 高校生が「参加したい」と感じられる、明るく親しみやすいトーンで
- 保護者にも必要な情報（交通・費用・付き添い可否等）が伝わること
- Web上で読みやすいよう、見出しと箇条書きを活用すること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**夏のオープンキャンパス2026 開催！**

〇〇大学の「リアル」を体験しよう！8月3日（日）・4日（月）、〇〇キャンパスでオープンキャンパスを開催します。模擬授業やキャンパスツアー、学食体験など、大学生活を丸ごと体感できるプログラムを多数ご用意。ぜひお気軽にお越しください。

**プログラム紹介**

- **学部別模擬授業**：全8学部の教員が、大学ならではの「学びの面白さ」をお届けします。1コマ30分、複数の授業を体験できます。

- **入試相談ブース**：入試制度や出願方法について、入試課スタッフが個別にお答えします。
- **奨学金・学生生活相談**：奨学金制度や学生寮、アルバイトなど、気になることを何でもご相談いただけます。
- **学食無料体験**：大学の学生食堂で人気メニューを無料で体験！

**FAQ**
