## IR活動の企画書を作成する
---

{{< alert context="info" text="**想定シーン**: IR担当職員が、学内の課題を明らかにするためのIR活動の企画書を作成する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたはInstitutional Research（以下、IR）の専門家です。
以下の要件に従い、IR活動の企画書を作成してください。

### 要件 ###
* 明らかにしたいこと：【変更箇所：教員の研究時間を阻害する要因を特定したい】
* 分析結果の活用目的：【変更箇所：研究力向上に向けた改善施策の立案に役立てたい】

### 企画書に含める要素 ###
1. 企画の背景と目的
2. 調査対象と調査方法
3. 分析の枠組み（分析項目・指標）
4. スケジュール（調査設計から報告までの工程）
5. 期待される成果と活用方法
6. 必要なリソース（人員・予算・ツール）

### 出力形式 ###
* 企画書本文：1000文字程度
* 概要：200文字程度
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**概要**
本企画は、教員の研究時間を阻害する要因を定量的・定性的に分析し、研究力向上に向けた実効性のある改善施策を立案するためのIR活動である。全教員を対象としたアンケート調査と時間利用調査を実施し、教育負担・会議・事務作業等の各要因が研究時間に与える影響を可視化する。調査結果は経営層への報告資料として活用し、具体的な業務改善策の提案につなげる。

**企画書**

**1. 背景と目的**
近年、本学教員の論文数・外部資金獲得件数に減少傾向がみられる。その背景として教員が十分な研究時間を確保できていない可能性が指摘されている。本企画では、研究時間の阻害要因を体系的に分析し、エビデンスに基づく改善施策を提案することを目的とする。

**2. 調査対象と方法**
- 対象：全専任教員（約300名）
- 方法：Webアンケート調査（所要時間15分程度）および2週間の時間利用ログ記録

**3. 分析の枠組み**

| 分析項目 | 指標例 |
|---------|--------|
| 教育負担 | 担当コマ数、授業準備時間 |
| 会議負担 | 月間会議時間、委員会数 |
| 事務作業 | 書類作成時間、学生対応時間 |
| 研究時間 | 週あたり研究充当時間 |

（以下、スケジュール・期待成果・リソースが続く）

## 学生調査データの傾向を分析する
---

{{< alert context="info" text="**想定シーン**: IR担当職員が、学生満足度調査の自由記述データを分析して改善点を抽出する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

{{< alert context="danger" text="個人情報（学籍番号・氏名・連絡先等）は必ず削除してからAIに入力してください。個人を特定できる情報を含む文章は絶対に入力しないでください。" />}}

```markdown
あなたはIRデータ分析の専門家です。
以下の学生調査の自由記述データを分析してください。

### 調査概要 ###
* 調査名：【変更箇所：2025年度学生満足度調査】
* 対象：【変更箇所：学部生全学年】
* 回答数：【変更箇所：1,200件】

### 分析してほしいこと ###
1. テーマ別分類：回答を主要なテーマに分類し、件数と割合を示す
2. 代表的回答の引用：各テーマの典型的な意見を2〜3件引用する
3. 感情傾向：肯定的・否定的・中立的な意見の割合を示す
4. 改善優先度の提案：影響度と改善可能性のマトリクスで優先順位を提案する

### 出力形式 ###
* テーマ別集計表（テーマ名・件数・割合・代表的意見）
* 改善優先度マトリクス（高影響×高実現可能性を最優先）
* 総括コメント（200文字程度）

### 自由記述データ ###
【ここに匿名化済みのデータを貼り付けてください】
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**テーマ別集計表**

| テーマ | 件数 | 割合 | 代表的意見 |
|-------|------|------|-----------|
| 授業内容 | 320件 | 26.7% | 「実践的な内容がもっとほしい」「グループワークが多すぎる」 |
| 施設・設備 | 280件 | 23.3% | 「自習スペースが不足している」「Wi-Fi環境を改善してほしい」 |
| 学生支援 | 200件 | 16.7% | 「履修相談の窓口時間を延長してほしい」 |
| キャリア支援 | 180件 | 15.0% | 「インターンシップの情報をもっと早く提供してほしい」 |
| その他 | 220件 | 18.3% | 「学食のメニューを増やしてほしい」 |

**改善優先度マトリクス**

| | 高実現可能性 | 低実現可能性 |
|---|---|---|
| **高影響** | Wi-Fi環境改善、履修相談窓口延長 | 自習スペース増設 |
| **低影響** | 学食メニュー拡充 | 大規模施設改修 |

**総括**: 授業内容と施設・設備に関する意見が全体の約50%を占めている。特にWi-Fi環境と自習スペースへの要望が強く、短期的にはWi-Fi増強と履修相談窓口の時間延長が費用対効果の高い施策と考えられる。
{{% /alert %}}


## エンロールメントデータを分析する
---

{{< alert context="info" text="**想定シーン**: 入試課・企画課職員が、入学者動向や歩留まり率を分析して募集戦略を検討する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学のエンロールメント・マネジメントの専門家です。
以下のデータをもとに、入学者動向と歩留まり率を分析してください。

### 分析対象 ###
* 期間：【変更箇所：過去5年間（2021〜2025年度）】
* 学部：【変更箇所：全学部（文学部・経済学部・理工学部・看護学部）】

### 分析に使用するデータ ###
* 志願者数・合格者数・入学者数（学部別・年度別）
* 歩留まり率（入学者数÷合格者数）
* 入試方式別の内訳（一般・推薦・総合型）

### 分析してほしいこと ###
1. 学部別の歩留まり率の推移と傾向
2. 歩留まり率に影響を与えている要因の仮説
3. 入試方式別の歩留まり率の比較
4. 競合大学との比較における本学の位置づけ（仮説ベース）
5. 歩留まり率改善のための具体的施策を3つ提案

### 出力形式 ###
表形式でデータを整理し、分析結果と施策提案を記述してください。

### データ ###
【ここにデータを貼り付けてください】
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**学部別歩留まり率の推移**

| 学部 | 2021年度 | 2022年度 | 2023年度 | 2024年度 | 2025年度 | 傾向 |
|------|---------|---------|---------|---------|---------|------|
| 文学部 | 32.1% | 30.5% | 29.8% | 28.3% | 27.0% | 低下傾向 |
| 経済学部 | 35.2% | 34.8% | 35.0% | 33.5% | 34.1% | 横ばい |
| 理工学部 | 28.5% | 27.3% | 26.1% | 25.8% | 24.5% | 低下傾向 |
| 看護学部 | 45.3% | 46.0% | 47.2% | 48.1% | 49.0% | 上昇傾向 |

**施策提案**
1. **合格者向けフォローアッププログラムの強化**: 合格発表後から入学手続き期限までの間に、在学生との交流イベントや学部紹介動画の配信を行い、入学意欲を高める
2. **入試方式の最適化**: 推薦・総合型選抜の歩留まり率が高い傾向を活かし、これらの定員比率を見直す
3. **データに基づく合格者数の精緻な設定**: 過去の歩留まりデータと外部環境を組み合わせた予測モデルを構築する
{{% /alert %}}

## ダミーデータを生成する
---

{{< alert context="info" text="**想定シーン**: IR担当職員が、分析手法の練習やプレゼンテーション用にダミーの学生データを作成する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたはデータ分析の専門家です。
IR業務の分析練習に使用するダミーデータを作成してください。

### データの仕様 ###
* データ件数：【変更箇所：10件】
* 対象：【変更箇所：架空の大学の学部生】
* 出力形式：CSV形式

### 含めるカラム ###
1. 学生ID（匿名のランダムID）
2. 学部（文学部・経済学部・理工学部・看護学部のいずれか）
3. 学年（1〜4年）
4. GPA（0.00〜4.00の範囲）
5. 出席率（%）
6. 満足度スコア（1〜5の5段階）
7. 課外活動参加（あり／なし）

### 条件 ###
* 実在する人物のデータではないことが明確であること
* 学部間でGPAや満足度に適度なばらつきを持たせること
* 分析練習に適したデータ分布にすること（極端な偏りを避ける）
```

## データの可視化方針を設計する
---

{{< alert context="info" text="**想定シーン**: IR担当職員が、分析結果をレポートにまとめる際に適切なグラフや図表の種類を選定する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたはデータ可視化の専門家です。
以下のデータセットに対して、最適なグラフ・図表の種類を提案してください。

### データセットの一覧 ###
1. 【変更箇所：学部別・年度別の入学者数推移（5年間×4学部）】
2. 【変更箇所：学生満足度調査の5段階回答の分布】
3. 【変更箇所：教員の業務時間の内訳（研究・教育・会議・事務の割合）】
4. 【変更箇所：GPAと出席率の関係（学生1,000名分）】
5. 【変更箇所：卒業生の就職先業種別割合】

### 提案してほしいこと ###
各データセットについて以下を回答してください。
1. 推奨するグラフの種類
2. その種類を選んだ根拠
3. 作成時の留意点（色使い・ラベル・凡例など）
4. 代替候補のグラフ種類

### 出力形式 ###
データセットごとに表形式で整理してください。
```

{{% alert icon="🤖" context="success" %}}
**出力例**

| データセット | 推奨グラフ | 根拠 | 留意点 | 代替候補 |
|------------|----------|------|--------|---------|
| 学部別・年度別入学者数推移 | 折れ線グラフ（学部別に色分け） | 時系列の変化を比較するのに適しており、複数系列の傾向を一目で把握できる | Y軸の起点を0にして誤解を防ぐ。学部ごとに区別しやすい配色を使用 | 積み上げ棒グラフ（全体の推移も把握したい場合） |
| 学生満足度の回答分布 | 水平棒グラフ（積み上げ） | 段階別の割合を直感的に比較できる。リッカートスケールの可視化に定番 | 肯定・否定をカラースケールで表現（青系→赤系） | 円グラフ（設問が1つの場合） |
| 教員の業務時間内訳 | 円グラフまたはドーナツチャート | 全体に占める割合を直感的に把握できる | カテゴリは4つまでが理想。割合のラベルを併記する | 帯グラフ（複数教員の比較時） |
| GPAと出席率の関係 | 散布図 | 2変数間の相関関係を視覚化するのに最適 | 回帰直線を追加して傾向を明示する。外れ値に注意 | ヒートマップ（データ密度を表現したい場合） |
| 卒業生の就職先業種別割合 | 横棒グラフ（降順） | カテゴリが多い場合でも見やすく、割合の大小を比較しやすい | 上位10業種に絞り「その他」でまとめる | ツリーマップ（階層構造を表現したい場合） |
{{% /alert %}}


## ベンチマーキング分析を支援する
---

{{< alert context="info" text="**想定シーン**: IR担当職員・企画課職員が、他大学との比較分析を設計し、自学の強み・弱みを客観的に把握する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学経営分析の専門家です。
他大学との比較分析（ベンチマーキング）の設計を支援してください。

### 本学の概要 ###
* 大学名：【変更箇所：A大学】
* 設置区分：【変更箇所：私立】
* 学部構成：【変更箇所：文学部・経済学部・理工学部・看護学部】
* 学生数：【変更箇所：約5,000名】

### 比較対象 ###
* 比較したい大学数：【変更箇所：5大学】
* 選定基準：【変更箇所：同規模・同地域・類似学部構成の大学】

### 設計してほしいこと ###
1. 比較項目の一覧（教育・研究・学生支援・財務・社会連携の各領域から）
2. 各項目のデータ収集方法（公開データの入手先を含む）
3. 分析フレームワーク（SWOT分析やレーダーチャートなど）
4. 比較結果の報告書の構成案

### 出力形式 ###
比較項目は表形式、分析フレームワークと報告書構成案はテキスト形式で出力してください。
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**比較項目一覧**

| 領域 | 比較項目 | データ収集方法 |
|------|---------|-------------|
| 教育 | ST比（学生教員比率） | 大学ポートレート、各大学公式サイト |
| 教育 | 退学率・休学率 | 学校基本調査、大学ポートレート |
| 研究 | 教員1人あたり論文数 | Scopus、Web of Science |
| 研究 | 外部資金獲得額 | 科研費データベース、各大学財務情報 |
| 学生支援 | 就職率・進学率 | 大学ポートレート、就職四季報 |
| 財務 | 教育研究経費比率 | 私立大学等経常費補助金データ |
| 社会連携 | 産学連携件数 | 各大学社会連携報告書 |

**分析フレームワーク**
- レーダーチャートで5大学の各指標を重ね合わせて可視化
- SWOT分析で本学の相対的な強み・弱み・機会・脅威を整理
- ギャップ分析でベンチマーク大学との差分を定量化

**報告書構成案**
1. 分析の目的と背景
2. 比較対象大学の選定理由
3. 比較データ一覧（表形式）
4. レーダーチャートによる総合比較
5. 領域別の詳細分析
6. SWOT分析の結果
7. 改善に向けた提言
{{% /alert %}}
