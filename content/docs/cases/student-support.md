## 学生面談の準備をする
---

{{< alert context="info" text="**対象: 学生支援課職員**<br>**想定シーン**: 学生支援課職員が、学生面談の前にシナリオ・質問リスト・留意点を準備する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは学生支援の専門家です。
学生との面談に向けた準備資料を作成してください。

### 面談の概要 ###
* 対象学生の状況：【変更箇所：成績低下が見られる2年生。前期GPAが1.8で、1年次の3.0から大幅に低下】
* 面談の目的：【変更箇所：成績低下の背景を把握し、適切な支援につなげる】
* 面談時間：【変更箇所：30分】

### 作成してほしいもの ###
1. 面談シナリオ（導入→本題→まとめの流れ）
2. 質問リスト（学生の状況を多角的に把握するための質問を5つ）
3. 留意点（面談時に気をつけるべきこと）
4. 想定される学生の反応と対応方針

### 条件 ###
* 学生を責めるような表現は避け、寄り添う姿勢を大切にすること
* 問題の原因を決めつけず、学生自身の言葉で語ってもらう方向で設計すること
* 必要に応じて学内の支援リソース（カウンセリング、学習支援など）への案内も含めること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**面談シナリオ**

| 段階 | 時間 | 内容 |
|------|------|------|
| 導入 | 5分 | 挨拶・面談の趣旨説明。「今日は○○さんのお話を聞かせていただきたくて」と安心感を与える |
| 本題 | 20分 | 質問リストに沿って学生の状況を把握。傾聴を中心に |
| まとめ | 5分 | 話の要約・今後の対応の確認・次回面談の日程調整 |

**質問リスト**
1. 最近の大学生活はいかがですか？（オープンクエスチョンで入る）
2. 授業で困っていることや気になっていることはありますか？
3. 大学以外の生活面で何か変化はありましたか？
4. 自分自身で「こうしたい」と思っていることはありますか？
5. 大学で「こういうサポートがあったら助かる」と感じることはありますか？

**留意点**
- 成績の数字を突きつけるのではなく、本人の実感を聞くところから始める
- 沈黙を恐れず、学生が考える時間を確保する
- メンタルヘルスの兆候がある場合は、学生相談室への橋渡しを丁寧に行う
{{% /alert %}}

## キャリア支援の助言を組み立てる
---

{{< alert context="info" text="**対象: 学生支援課職員**<br>**想定シーン**: キャリアセンター職員が、学生のキャリア相談に備えて業界研究・自己分析の観点を整理する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたはキャリアコンサルタントの知見を持つ大学キャリアセンターのアドバイザーです。
以下の学生情報をもとに、キャリア相談の準備をしてください。

### 学生の情報 ###
* 学部・学年：【変更箇所：経済学部3年生】
* 関心のある分野：【変更箇所：金融業界とIT業界に興味があるが絞り切れていない】
* 強み・経験：【変更箇所：ゼミでデータ分析を経験。アルバイトで接客業を3年継続】
* 悩み：【変更箇所：自分に合った業界がわからない。大企業とベンチャーどちらがよいかも迷っている】

### 作成してほしいもの ###
1. 学生の強みと経験を活かせるキャリアパスの提案（3つ）
2. 各キャリアパスで必要な準備事項
3. 自己分析を深めるための問いかけ（3つ）
4. 次のアクションプラン（1ヶ月以内にできること）

### 条件 ###
* 特定の企業名は出さず、業界・職種レベルで提案すること
* 学生の主体的な意思決定を尊重する書き方にすること
* 大学のキャリアセンターで活用できるリソース（模擬面接、OB訪問支援等）への案内も含めること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**キャリアパスの提案**

| パス | 業界・職種 | 活かせる強み | 準備事項 |
|-----|----------|------------|---------|
| パス1 | 金融業界・データ分析職 | ゼミのデータ分析経験、数値を扱う素養 | 金融関連資格（FP等）の学習、Pythonの基礎習得 |
| パス2 | IT業界・ビジネスコンサルタント | データ分析力と接客で培ったコミュニケーション力 | IT業界研究セミナーへの参加、ケーススタディの練習 |
| パス3 | フィンテック企業・企画職 | 金融とITの両方への関心、分析力 | フィンテック業界の動向調査、インターンシップへの応募 |

**自己分析を深める問いかけ**
1. 「これまでの経験で、時間を忘れるほど集中できた瞬間はどんな場面でしたか？」
2. 「チームで活動するとき、自分はどんな役割を担うことが多いですか？」
3. 「5年後の自分を想像したとき、どんな働き方をしていたいですか？」

**次のアクションプラン**
- 今週中：キャリアセンターのOB訪問支援に申し込む（金融・IT各1名）
- 2週間以内：業界研究セミナーに1つ参加する
- 1ヶ月以内：模擬面接を1回受けて、自己PRの言語化を進める
{{% /alert %}}


## 合理的配慮の対応案を検討する
---

{{< alert context="info" text="**対象: 学生支援課職員**<br>**想定シーン**: 障害学生支援室の職員が、障害のある学生への配慮事項を整理し、対応案を検討する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

{{< alert context="danger" text="AIの提案はあくまで参考情報です。個別の学生への合理的配慮は、本人との対話・専門家の助言・関係部署との連携を通じて、担当者が責任を持って判断してください。" />}}

```markdown
あなたは大学における障害学生支援の知見を持つアドバイザーです。
以下の状況に対する合理的配慮の対応案を検討してください。

### 学生の状況 ###
* 障害の種別：【変更箇所：視覚障害（弱視・矯正視力0.1程度）】
* 学部・学年：【変更箇所：文学部2年生】
* 本人の要望：【変更箇所：授業資料の拡大、試験時間の延長、座席の配慮】

### 検討してほしいこと ###
1. 授業場面での具体的配慮案
2. 試験場面での具体的配慮案
3. 施設・設備面での配慮案
4. 教職員への周知事項
5. 配慮の実施にあたっての留意点

### 条件 ###
* 障害者差別解消法に基づく合理的配慮の考え方を踏まえること
* 本人の意思と自己決定を尊重する視点を持つこと
* 過度な負担とならない範囲で実現可能な案を中心に提案すること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**具体的配慮案**

| 場面 | 配慮内容 | 実施方法 |
|------|---------|---------|
| 授業 | 資料の拡大提供 | 配付資料をA3サイズに拡大、またはPDFデータで事前配布 |
| 授業 | 座席の配慮 | 最前列中央の座席を確保し、板書が見やすい位置を保証 |
| 授業 | 板書の代替手段 | 授業後にスライドデータを提供、またはノートテイカーを配置 |
| 試験 | 時間延長 | 通常の1.3倍の試験時間を付与（60分→78分） |
| 試験 | 問題用紙の拡大 | A3サイズへの拡大印刷、フォントサイズ14pt以上 |
| 施設 | 学習環境の整備 | 拡大読書器の設置、照明環境の調整 |

**留意点**
- 配慮内容は学期ごとに本人と面談し、必要に応じて見直す
- 授業担当教員への説明は、本人の同意を得た範囲で行う
- 配慮により他の学生との公平性が損なわれないよう配慮内容を設計する
{{% /alert %}}


## 退学・休学相談への対応を準備する
---

{{< alert context="info" text="**対象: 学生支援課職員**<br>**想定シーン**: 学生課職員が、退学・休学を検討している学生との面談に向けて対応フローや声かけを準備する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは学生支援の専門家です。
退学・休学を検討している学生への対応準備を手伝ってください。

### 想定される状況 ###
* 学生の状況：【変更箇所：3年生。経済的な理由と学業への意欲低下により退学を検討中】
* 相談の経緯：【変更箇所：教務窓口に本人が来訪し、退学届の用紙を求めた】

### 作成してほしいもの ###
1. 段階的な対応フロー（初回対応→情報収集→支援提案→フォローアップ）
2. 面談で使える声かけの例（寄り添う姿勢を重視）
3. 確認すべき事項のチェックリスト
4. 活用できる学内支援リソースの整理
5. 想定Q&A（学生からよくある質問と対応例を3つ）

### 条件 ###
* 退学を引き止めることが目的ではなく、学生が十分な情報を得た上で判断できるようにすることが目的
* 経済支援・学習支援・メンタルサポートの各側面からの選択肢を提示すること
* 休学という選択肢のメリット・デメリットも説明に含めること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**段階的対応フロー**

| 段階 | 対応内容 | ポイント |
|------|---------|---------|
| 1. 初回対応 | 来訪目的の確認。退学届をすぐに渡すのではなく、面談の機会を設ける | 「お話を聞かせてもらえますか」と穏やかに提案 |
| 2. 情報収集 | 退学を検討する背景を丁寧にヒアリング | 経済面・学業面・心理面を網羅的に確認 |
| 3. 支援提案 | 利用可能な支援制度・選択肢を提示 | 奨学金・授業料減免・休学制度の説明 |
| 4. フォローアップ | 1週間後に連絡。決定を急がせない | 「いつでも相談に来てくださいね」と伝える |

**想定Q&A**

Q1. 「退学届を出したら、すぐに退学になりますか？」
A1. 「退学届の提出後、教授会での審議を経て正式に決定されます。通常2〜3週間かかります。提出後でも審議前であれば取り下げることも可能です。」

Q2. 「休学すると学費はかかりますか？」
A2. 「本学では休学期間中の授業料は免除されますが、在籍料として年間○万円が必要です。休学期間は最長2年間で、復学時の手続きもサポートします。」

Q3. 「退学した場合、これまでの単位はどうなりますか？」
A3. 「取得済みの単位は成績証明書に記録されます。将来別の大学に編入学する場合、一部の単位が認定される可能性があります。」
{{% /alert %}}


## オリエンテーション資料を作成する
---

{{< alert context="info" text="**対象: 教務課職員**<br>**想定シーン**: 教務課職員が、新入生向けのオリエンテーション資料（履修登録の手順ガイドなど）を作成する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学の教務課職員の業務をサポートする文書作成の専門家です。
新入生向けのオリエンテーション資料を作成してください。

### 資料の概要 ###
* テーマ：【変更箇所：履修登録の手順ガイド】
* 対象：【変更箇所：学部新入生（1年生）】
* 配布場所：【変更箇所：新入生オリエンテーション配付資料、学生ポータルにPDF掲載】

### 資料に含める内容 ###
1. 履修登録の全体スケジュール
2. 履修登録システムへのログイン方法
3. 科目の選び方のポイント（必修・選択必修・自由選択の違い）
4. 履修登録の具体的な手順（ステップ形式で）
5. よくある質問（FAQ）を3つ
6. 困ったときの問い合わせ先

### 条件 ###
* 大学の仕組みに不慣れな新入生が読むことを想定し、専門用語には説明を添えること
* 手順はステップごとに番号を振り、1画面ずつ操作を説明するイメージで
* 親しみやすく、かつ正確なトーンで書くこと
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**履修登録の手順ガイド**

**1. 全体スケジュール**

| 日程 | 内容 |
|------|------|
| 4/1〜4/3 | 学生ポータルへのログイン確認期間 |
| 4/4〜4/5 | 履修ガイダンス（学部別） |
| 4/7〜4/11 | 履修登録期間（Web登録） |
| 4/14〜4/18 | 履修修正期間 |
| 4/21 | 履修確定・時間割公開 |

**2. 科目の選び方のポイント**
- **必修科目**: 卒業に必ず必要な科目です。自動登録される場合もあるので、ガイダンスで確認しましょう
- **選択必修科目**: 指定されたグループの中から一定数を選んで履修する科目です
- **自由選択科目**: 自分の興味に合わせて自由に選べる科目です。他学部の科目を履修できる場合もあります

**3. 履修登録の手順**
- Step 1: 学生ポータル（https://portal.xxx.ac.jp）にアクセスし、学籍番号と初期パスワードでログイン
- Step 2: メニューから「履修登録」を選択
- Step 3: 曜日・時限ごとに科目を検索し、登録ボタンを押す
- Step 4: 登録内容を確認し、「確定」ボタンを押す
- Step 5: 登録完了画面を必ずスクリーンショットで保存する

## 奨学金ガイダンス資料を作成する
---

{{< alert context="info" text="**対象: 学生支援課職員**<br>**想定シーン**: 学生課職員が、奨学金制度の説明資料を作成し、ガイダンスで使用する" />}}

「変更箇所」を自身の状況に合わせて変更してください。

```markdown
あなたは大学の学生課職員の業務をサポートする文書作成の専門家です。
奨学金制度の説明資料を作成してください。

### 資料の概要 ###
* テーマ：【変更箇所：奨学金制度の概要と申請手続きガイド】
* 対象：【変更箇所：学部生全学年（特に新入生・2年生）】
* 使用場面：【変更箇所：奨学金ガイダンス配付資料】

### 資料に含める内容 ###
1. 奨学金制度の全体像（日本学生支援機構・大学独自・民間団体の分類）
2. 日本学生支援機構奨学金の種類と比較表（第一種・第二種・給付型）
3. 申請から採用までの流れ（ステップ形式）
4. 申請時に必要な書類の一覧
5. 返還に関する基本情報
6. よくある質問（FAQ）を3つ

### 条件 ###
* 制度の正確性を重視しつつ、学生が理解しやすい平易な表現で書くこと
* 金額や条件は一般的な情報として記載し、「詳細は最新の募集要項を確認してください」と注記すること
* 返還が必要な奨学金と不要な奨学金の違いを明確にすること
```

{{% alert icon="🤖" context="success" %}}
**出力例**

**日本学生支援機構奨学金の比較表**

| 項目 | 給付型 | 第一種（貸与） | 第二種（貸与） |
|------|--------|-------------|-------------|
| 返還 | 不要 | 必要（無利子） | 必要（有利子） |
| 月額（自宅通学） | 12,800円〜38,300円 | 20,000円〜54,000円 | 20,000円〜120,000円 |
| 月額（自宅外通学） | 25,300円〜75,800円 | 20,000円〜64,000円 | 20,000円〜120,000円 |
| 成績基準 | 学習意欲があること | 平均水準以上 | 平均水準以上 |
| 家計基準 | 住民税非課税世帯等 | 一定の収入基準以下 | 第一種より緩やか |
| 併用 | 第一種と併用可 | 給付型・第二種と併用可 | 給付型・第一種と併用可 |

※金額は2025年度の一般的な情報です。最新の募集要項で必ず確認してください。

**申請の流れ**
1. 奨学金ガイダンスに参加する
2. スカラネット（Web申請システム）で申込情報を入力する
3. 必要書類（マイナンバー提出書類、収入に関する証明書等）を学生課に提出する
4. 日本学生支援機構による選考
5. 採用通知の受領・返還誓約書の提出
6. 奨学金の振込開始

**FAQ**
Q1. 「奨学金はいつから振り込まれますか？」
A1. 「4月に申請した場合、通常6〜7月に初回振込があります。採用決定後に4・5月分もまとめて振り込まれます。」

Q2. 「成績が下がったら奨学金は止まりますか？」
A2. 「適格認定（年1回の審査）で学業成績が基準を下回ると、警告・停止・廃止の措置がとられる場合があります。」

Q3. 「途中で辞退することはできますか？」
A3. 「はい、可能です。学生課に申し出てください。貸与型の場合、辞退までに借りた分の返還は必要です。」
{{% /alert %}}
