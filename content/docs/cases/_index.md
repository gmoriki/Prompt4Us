このセクションでは、大学業務で実際に活用できるプロンプトの実例を紹介します。
文書作成、情報収集、仮説の整理、アイデア出しなど、目的に応じたプロンプトをご覧ください。

各プロンプトには**想定シーン**（誰が・どんな場面で使うか）と**出力例**（AIがどのように回答するか）を掲載しています。プロンプト中の「変更箇所」をご自身の状況に合わせて書き換えるだけで、すぐに活用できます。

---

### カテゴリ一覧

| カテゴリ | 概要 |
|---------|------|
| **文書作成** | 告知文・企画書・議事録・引継書・報告書など、大学業務で必要な文書の作成を支援 |
| **メール対応** | 学内外へのメール文面の作成・校正・敬語チェック |
| **授業設計** | シラバス作成・授業計画・教材設計のたたき台づくり |
| **評価** | ルーブリック作成・成績評価基準の設計 |
| **研究支援** | 論文要約・文献レビュー・研究計画のブレスト |
| **IR** | 学内データの分析設計・可視化・レポート作成 |
| **学生対応** | 学生向け案内文・相談対応メモ・FAQ作成 |
| **国際交流** | 英文レター・多言語対応・海外機関との連絡文 |
| **規程・認証** | 規程文案の作成・認証評価の自己点検資料作成 |
| **広報** | プレスリリース・SNS投稿文・キャッチコピー作成 |
| **アイデア** | ブレインストーミング・企画立案・課題整理 |
| **テキスト変換** | 翻訳・要約・文体変換・書式整形 |
| **情報収集** | 調査・リサーチ・データ収集の効率化 |
| **コーディング** | Excel関数・簡易スクリプト・データ処理の自動化 |
| **発展プロンプト** | メタプロンプト・Chain of Thoughtなど高度な活用法 |

AIに複数ステップの仕事を任せる「エージェント」的な活用については、[AIエージェントとタスク設計]({{% relref "/docs/intermediate/advanced-use/agent-task-design" %}})で考え方を紹介しています。
